1. Clone the repository to your local machine.

2. Open a terminal window and navigate to the project directory.

3. Ensure that you have the necessary C++ compiler installed.

4. Compile the program using the provided makefile by running the following command:

    make all

5. After successful compilation, run the program by executing the following command:

    make run

6. To clean up the compiled object files and the executable, run the following command:

    make clean

Representation of the Tree

I represent the B-Tree using a node-based approach. Each node in the tree contains a vector of keys and a vector of child nodes. This design allows for effective management and traversal of the tree structure.
